Box Model

Layout

  • Es la geometría de los elementos que define: Tamaño, la separación entre elementos adyascentes, la posición en la que se dibujará el elemento
  • Propiedades: width y height, el ancho y el alto

Elementos de Línea (inline) y bloque (block)

Características inline

  1. No crean lineas para cada elemento
  2. No se puede definir el ancho y el alto
  3. el ancho se ajista de acuerdo al contenido

Características block

  1. Crean una linea para cada elemento
  2. Ocupan todo el espacio horizontal disponible
  3. Tienen un ancho y alto
